# Generate Your Own Scripts

These files contain **prompts, not scripts**. Paste any of them into Claude, Gemini, ChatGPT, or another AI, fill in your details where marked, and the AI will generate a script tailored to your exact setup.

## Available Prompts

1. **status-line.md** — Generate a live status line showing context usage, tokens, git branch, LM Studio model, or other metrics
2. **model-manager.md** — Generate a model management script for LM Studio, Ollama, or other local LLM platforms
3. **usage-logger.md** — Generate a script to track and log LLM usage to SQLite (proves ROI of local inference)
4. **context-backup.md** — Generate session save/restore scripts to checkpoint AI coding session state
5. **process-cleanup.md** — Generate a cleanup script for orphaned background processes from AI tools

Each prompt asks you to specify your setup (tool, OS, preferences) and outputs production-ready code.
